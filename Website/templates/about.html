About EVInsight

A data analytics project leveraging Tableau to explore electric vehicle charging patterns and range performance.

⚡

Project Overview

This project uses Flask to create a simple, elegant web interface that embeds Tableau Dashboards directly into the web app. Using the Tableau Public API, interactive visualizations are rendered seamlessly, allowing users to explore EV charge and range data without leaving the site.

The visualization focuses on understanding patterns in how electric vehicles charge — including factors like charging time, battery capacity, and real-world range — providing valuable insights for EV owners, manufacturers, and policy makers.

🛠️

Technology Stack

🐍 Python / Flask 📊 Tableau Public 🎨 HTML / CSS / JS 🖥️ Jinja2 Templates
🎯

Key Features

  • ⚡ Live embedded Tableau visualization with full interactivity
  • 🔋 EV charge duration and battery capacity analysis
  • 📍 Real-world range vs. rated range performance tracking
  • 🚗 Multi-model comparison across EV brands and categories
  • 📱 Responsive design optimized for all screen sizes
  • 🌙 Dark mode premium interface with glassmorphism styling
